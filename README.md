# Prowider Mini Lead Distribution System

This repository contains a Next.js App Router application for fair lead allocation across providers.

## Tech stack

- Next.js App Router
- TypeScript
- PostgreSQL
- Prisma ORM
- Tailwind CSS

## Next steps

1. Define a complete Prisma schema for the lead distribution domain.
2. Create seed data and provider allocation state.
